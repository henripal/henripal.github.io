Bayesian statistics is an incredibly powerful tool for data analysis! I'm currently reading J. Kruschke's "Doing Bayesian Data Analysis with R" and loving it. All the examples in the book are in R, JAGS and STAN, so I decided to take notes in a Jupyter notebook using Python and PyMC.

My notes won't go over all the examples, but I try to explain every step carefully, especially as it pertains to Python or PyMC things! If you want a more exhaustive look at all the examples, directly translated in Python, see Osvaldo Martin's github repository.
